# Illenium Appearance - Advanced Customization System

**Illenium Appearance** is a complete, feature-rich replacement for standard clothing resources, designed for **QB-Core**, **ESX**, and **Qbox**. It brings modern UI design, robust functionality, and extensive customization options to your FiveM server.

This resource has been significantly enhanced with a **brand new, modern Svelte-based UI**, featuring hexagon navigation, smooth animations, and a sleek dark aesthetic.

## üì• Installation

1. **Download** the latest release.
2. **Extract** to your resources folder.
3. **Add** `ensure illenium-appearance` to your `server.cfg`.
4. **Disable** conflicting resources:
   - `qb-clothing` (for QB-Core)
   - `fivem-appearance` (if installed previously)
   - `esx_skin` / `skinchanger` (for ESX)

---

## üì¶ Dependencies

- **[ox_lib](https://github.com/overextended/ox_lib)** (Required for all frameworks)
- **Framework Specifics**:
  - `qb-core` (for QB Servers)
  - `es_extended` (for ESX Servers)
  - `ox_core` (for Ox Servers)

## üåç Localization

Ready to use out of the box with multiple languages supported.
To change the language, set the convar in your `server.cfg`:

```cfg
setr illenium-appearance:locale "en"
# Available: en, de, fr, es, etc.
```
